# seqpaint examples

Curated sample outputs illustrating what seqpaint can do. Each subfolder is a self-contained mini-tutorial:

| Folder | What it shows |
|---|---|
| [`layouts/`](layouts/) | What the three layouts (raster / Hilbert / Z-order) actually look like, side by side on the same data |
| [`palettes/`](palettes/) | All four named palettes rendering the same nucleic-acid sequence |
| [`hilbert/`](hilbert/) | When Hilbert layout actually helps — GC content and k-mer coloring compared raster vs Hilbert |
| [`single/`](single/) | Individual sequence renderings: 16S rRNA, human mitochondrion, lambda phage |
| [`comparative/`](comparative/) | Two-input modes: dotplot, diff, and stacked multi-FASTA |
| [`formats/`](formats/) | SVG and HTML outputs (same PNG feature set, different backends) |

All PNGs in these folders were generated by the seqpaint CLI against sequences fetched from NCBI via `--accession`. The raw sequences are not bundled; reproduction commands are listed in each folder's README.
